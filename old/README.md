# 🚀 Priority Todo - Nurture Your Productivity

A high-performance, gamified dashboard for task management built with **React 19**, **Vite**, and **Tailwind CSS**. This app turns your daily goals into a rewarding experience—watch your digital plant thrive as you check off your tasks!

![Speed](https://img.shields.io/badge/Build-Vite%20%2B%20Lightning%20Fast-blue)
![Privacy](https://img.shields.io/badge/Data-100%25%20Local-emerald)

## 🌟 The "E-Tree" Experience
- **🌿 Gamified Growth**: Earn 10ml, 20ml, or 40ml of water based on task priority (Inspired by MS Edge Virtual Plant). Level up your plant from a tiny sprout to a blooming masterpiece.
- **📈 Persistence Tracker**: A dedicated streak counter monitors your daily growth momentum.
- **🔋 Energy History**: A scrollable 30-day timeline tracks your historical watering progress.

## ✨ Core Features
- **💎 Premium Dashboard**: A sleek, 2-column layout with a modern "Snow Lake" aesthetic and smooth glassmorphism.
- **📅 Smart Deadlines**: Visual overdue alerts for tasks that need immediate attention.
- **🔍 Tactical Sorting**: Filter by category (Work, Study, Personal) and sort by Priority or Due Date in a single click.
- **💾 Local Persistence**: All data is stored in `localStorage`—no login, no backend, no latency.
- **🧹 Instant Cleanup**: A dedicated "Clean Up" tool to archive completed tasks and keep your workspace clear.

## 🛠️ Tech Stack
- **React 19** & **Framer Motion** (Smooth transitions & animations)
- **Tailwind CSS** (Custom utility-first styling)
- **Canvas Confetti** (Rewarding celebrations)
- **React Icons** (Modern iconography)

## 🚀 Installation & Setup

```bash
# 1. Clone the repo
git clone https://github.com/siddhali24/priority-todo.git

# 2. Enter project
cd priority-todo/frontend

# 3. Install
npm install

# 4. Launch
npm run dev
```
